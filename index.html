StreamerOCR Documentation

Overview

StreamerOCR is a lightweight desktop application that allows you to perform real-time OCR (Optical Character Recognition) on selected screen regions and convert the text to speech. It's particularly useful for streamers, content creators, and users who need to quickly extract and read text from their screen.

Features

  • Screen region selection with visual feedback
  • Real-time OCR processing using Tesseract
  • Text-to-Speech conversion
  • Global unique hotkeys for quick access
  • Region persistence between sessions

Keyboard Shortcuts

  • Alt + Shift + Space - Process the selected region
  • Alt + Shift + M - Select a new region
  • Alt + Shift + N - Exit application

Usage

  1. Launch the StreamerOCR application
  2. Click the "Select Region" button or press Alt + Shift + to select a screen region
  3. Click and drag to select the desired area
  4. Press Alt + Shift + Space to process the selected region
  5. The application will read aloud any text found in the region

Tips

  • For best OCR results, ensure the text in the selected region is clear and well-contrasted
  • The selected region is saved between sessions
  • If no text is found in the region, a notification will appear
  • Press Escape while selecting a region to cancel the selection

Requirements

  • Windows 10 or later
  • Tesseract OCR engine (included in the installation)
  • Python 3.8 or later (if running from source)
