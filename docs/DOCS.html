1. Overview

StreamerOCR is a powerful screen text recognition tool designed for streamers and content creators. It allows you to:

  • Select and capture specific regions of your screen
  • Perform OCR (Optical Character Recognition) on the captured regions
  • Convert recognized text to speech
  • Save and manage multiple capture regions
  • Work across multiple monitors

2. Setup & Installation

⚠️ Important: Tesseract OCR Required

StreamerOCR requires Tesseract OCR to be installed separately. This is not included in the package.

  1. Download Tesseract OCR from: UB-Mannheim Tesseract Installer
  2. Run the installer
  3. Add Tesseract to your system PATH (default: C:\Program Files\Tesseract-OCR)
  4. Restart your computer after installation

Without Tesseract OCR installed, StreamerOCR will not function properly.

Prerequisites

  1. Python 3.8 or higher
  2. Tesseract OCR (see above)
  3. Windows operating system

Installation Steps


# Clone the repository
git clone [repository-url]
cd StreamerOCR

# Create and activate virtual environment
python -m venv venv
.\venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

Note:

Make sure Tesseract OCR is installed and accessible in your system PATH before running the application.

3. Usage Guide

Running the Application

python src/main.py

Hotkeys

  • Alt+Shift+Space: Process selected region
  • Alt+Shift+M: Select new region
  • Alt+Shift+N: Exit application

Region Management

  • Click "Select Region" to choose a screen area
  • Use the region list to manage saved regions
  • Rename or delete regions as needed
  • Regions are automatically saved between sessions

Multi-Monitor Support:

The application supports multiple monitors. When selecting a region, you can choose which monitor to capture from using the monitor dropdown.
