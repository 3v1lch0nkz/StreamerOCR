1. Setup, Installation, and Usage

Installation


# Clone the repository
git clone [repository-url]
cd StreamerOCR

# Create and activate virtual environment
python -m venv venv
.\venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac

# Install dependencies
pip install -r requirements.txt

Configuration

The application requires Tesseract OCR to be installed on your system:

  1. Download and install Tesseract OCR (see Requirements section)
  2. Ensure Tesseract is in your system PATH or in one of the default locations:
    • C:\Program Files\Tesseract-OCR
    • C:\Program Files (x86)\Tesseract-OCR

Usage

To run the application:

python src/main.py

Hotkeys:

  • Ctrl+Shift+O: Process selected region
  • Ctrl+Shift+K: Show/restore window
  • Ctrl+Shift+M: Exit application

Running Tests

The project includes several test suites:


# Run all tests
python run_tests.bat

# Run individual test suites
python tests/test_basic.py      # Basic functionality tests
python tests/test_integration.py # Integration tests
python tests/test_region.py     # Region selection tests
python tests/test_ocr.py        # OCR functionality tests

  • TTS Speaker (src/tts/speaker.py)

    • Voice configuration

3. External Requirements

  • Tesseract OCR

    Required for text recognition. Not bundled with the application.

    Download from: UB-Mannheim Tesseract Installer

    • Version: 5.0.0 or higher recommended
    • Installation path should be added to system PATH
    • Default paths:
      • Windows: C:\Program Files\Tesseract-OCR
      • Linux: /usr/bin/tesseract

  • Python Dependencies

    These are automatically installed via requirements.txt:

    • PyQt5 (5.15.9) - GUI framework
    • pytesseract (0.3.10) - Tesseract OCR wrapper
    • Pillow (9.5.0) - Image processing
    • pyttsx3 (2.90) - Text-to-speech engine
    • keyboard (0.13.5) - Hotkey handling
    • Note: Some features may require additional system permissions:
    • Screen capture functionality requires display access
    • Text-to-speech may require audio output permissions
    • Global hotkeys may require administrative privileges on some systems
